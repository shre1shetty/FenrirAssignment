# Frontend Design Challenge – Fenrir Security

## Tech Stack

- React (Vite)
- React Router
- Tailwind CSS v4
- Mock JSON data

### Folder Structure

- components/ – Reusable UI components
- pages/ – Screen-level components
- Layout/ – Layout components
- utils/ – utility files

##### Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

#### Known Limitations

The application has been optimized for desktop view (1280px+).
Mobile responsiveness (375px) would be the next step for refinement, but due to the 24-hour constraint, focus was placed on pixel-accurate desktop implementation, theme consistency, and full interactivity.
